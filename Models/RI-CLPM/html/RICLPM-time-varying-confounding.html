Lets investigate the effect of a simple time-varying confounder. Here Im mostly intersting in creating bias for the x -> y cross-lagged effects.

In the DAG Ive added a time-varying variable thas causes both \(x_i\) and \(y_{i+1}\).

## Time varying confounding
# 3 time points with compound symmetry covariance
add_time_varying_confounding <- function(
    data,
    mu = c(0, 0, 0),
    SD = 2, 
    corr = 0.8) {

        n <- nrow(data)
        cor_C <- SD * 2 * corr
        sigmaC <- matrix(
            c(
                SD^2, cor_C, cor_C,
                cor_C, SD^2, cor_C,
                cor_C, cor_C, SD^2
            ),
            nrow = 3, ncol = 3
        )
        C <- MASS::mvrnorm(n, mu, sigmaC)

        data <- data %>% 
        mutate(
            c1 = C[, 1],
            c2 = C[, 2],
            c3 = C[, 3],
            x1 = x1 + c1,
            x2 = x2 + c2,
            x3 = x3 + c3,
            y1 = y1,
            y2 = y2 + c1,
            y3 = y3 + c2
        )

        data
}

Adjust for time-varying confounders

Lets see if we adjust for confounders and get correct results. Ive simply ddded this to the lavaan RICLPM:

  # adjust for time-varying confounders
  x1 ~ c1
  x2 ~ c2
  x3 ~ c3 
  y2 ~ c1
  y3 ~ c2
